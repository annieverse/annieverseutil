# Contributing to annieverseUtil

Thank you for your interest in contributing! Your help is greatly appreciated.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Clone your fork** and make your changes.
3. **Write clear, concise commit messages.**
4. **Add tests** for new features or bug fixes.
5. **Run tests** to ensure everything passes: `npm run jest`
6. **Submit a pull request** with a clear description of your changes.

## Code Style
- Follow the existing code style and structure.
- Use ESLint to check your code: `npm run test`
- Write descriptive comments where necessary.

## Reporting Issues
- Use the GitHub Issues tab to report bugs or request features.
- Provide as much detail as possible (steps to reproduce, expected behavior, etc).

## Community
- Be respectful and inclusive in all interactions.
- See our [Code of Conduct](CODE_OF_CONDUCT.md).

Thank you for helping make annieverseUtil better!
